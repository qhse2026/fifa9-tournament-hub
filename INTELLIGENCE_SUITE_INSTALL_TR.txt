A) SUPABASE — YALNIZCA TAHMİN LİGİ İÇİN
----------------------------------------
1. Supabase projenizi açın.
2. SQL Editor > New Query seçin.
3. supabase/intelligence_feature_v15.sql dosyasının tamamını yapıştırın.
4. Run düğmesine basın.
5. Başarı mesajını kontrol edin.

Bu SQL mevcut maçları, chat mesajlarını ve oyuncu hesaplarını silmez.
Mevcut v12 Simple PIN Chat kurulumu gereklidir.

TAHMİN LİGİ AKIŞI
-----------------
- Oyuncu önce Sohbet sekmesinde kayıtlı adını seçip PIN ile giriş yapar.
- Zekâ Merkezi > Tahmin Ligi sekmesine gider.
- Maç başlamadan skor tahminini kaydeder.
- Aynı maç için tahminini maç canlıya alınmadan önce güncelleyebilir.
- Maç sonucu kaydedildiğinde puan otomatik hesaplanır.

PUANLAMA
--------
Tam skor: 5 puan
Doğru sonuç: 3 puan
Doğru gol farkı: +1 puan
Sürpriz sonucu bilme: +1 puan

---

karşılaştırma amacı taşır. Gerçek bahis, para yatırma veya ödeme işlevi yoktur.
