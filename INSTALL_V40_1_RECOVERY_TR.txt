1) SUPABASE
Supabase > SQL Editor'da şu dosyanın tamamını çalıştır:
  fifa_v40_1_player_access_and_poll.sql

SQL mevcut tournament_state içindeki current.participants listesini otomatik olarak
fifa09_registered_players tablosuna aktarır.

Supabase > Authentication > Providers > Email bölümünde Email provider açık olmalıdır.
Hesapların anında açılmasını istiyorsan "Confirm email" seçeneğini kapatabilirsin.
Confirm email açık kalırsa kullanıcı e-posta doğrulamasından sonra giriş yapar.

2) GITHUB

  fifa-player-access-v40-1.css
  fifa-player-access-v40-1.js

3) INDEX.HTML

- Ctrl + F5 yap.
- Hâlâ eski sayfa gelirse DevTools > Application > Service Workers > Unregister,
  ardından Storage > Clear site data.
