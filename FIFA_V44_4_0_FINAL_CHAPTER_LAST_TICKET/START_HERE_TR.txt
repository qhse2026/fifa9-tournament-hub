FIFA V44.4.0 — FINAL CHAPTER SON BİLET

GITHUB'A YÜKLENECEK DOSYALAR
ZIP içindeki github klasöründen yalnızca şu üç dosyayı repository köküne yükleyin:

1. app.js
2. styles.css
3. service-worker.js

Aynı adlı mevcut dosyaların üzerine yazın. Diğer dosyaları silmeyin.

YÜKLEME SONRASI
1. GitHub Pages deploy işleminin tamamlanmasını bekleyin.
2. Siteyi açın.
3. Ctrl + Shift + R yapın.
4. Eski format görünürse tarayıcı geliştirici araçlarından Application > Service Workers > Unregister yapın ve tekrar yenileyin.
