GITHUB KURULUMU
1. ZIP dosyasını açın.
2. github klasörünün içeriğini repository köküne yükleyin.
3. Aynı adlı dosyaları Replace / Overwrite edin.
4. data klasöründeki iki JSON dosyasını kendi data klasörünüze yükleyin.
5. GitHub Pages deploy tamamlandıktan sonra Ctrl+F5 yapın.
6. Gerekirse tarayıcı Site Settings > Clear data ve Service Worker unregister işlemi uygulayın.
